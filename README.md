# 🔐 SecureAuth – Login & Registration System

## Overview

SecureAuth is a front-end authentication system that demonstrates user registration, login, session handling, and protected page access using Local Storage.

## Features

- User Registration
- User Login
- Logout
- Protected Dashboard
- Form Validation
- Local Storage Authentication
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Note

This project uses Local Storage to simulate authentication for learning purposes. In a real-world application, authentication should be implemented using a secure backend and password hashing.

## How to Run

1. Open the project folder.
2. Run `index.html` using Live Server.
